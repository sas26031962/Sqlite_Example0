# Sqlite_Example0
Правила формирования файла ExternalText.txt
1. Первым идёт имя автора, оно отделяется от остального текста конструкцией 
" - ".
2. После констукции " - " идёт название серии, оно отделяется от остального 
текста конструкцией ".".
3. После констукции "." идёт номер тома, он начинается с символа '#' и содержит 
строку, которую следует преобразовать в номер
